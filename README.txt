CyberSecure Academy — Full package
=================================

Instrukcja:
1. Wypakuj paczkę.
2. W pliku js/firebase-config.js wklej konfigurację Firebase (jeśli chcesz używać Auth/Firestore/Storage).
3. Umieść pełny kod aplikacji (jeśli chcesz zastąpić index.html).
4. Uruchom lokalnie: otwórz index.html w przeglądarce.
5. Deployment:
   - Firebase Hosting: zainstaluj firebase-tools i wykonaj firebase init + firebase deploy.
   - Netlify / Vercel: przeciągnij folder do panelu lub połącz repo.
Uwaga: Pliki zawierają przykładowe fiszki (1000) w flashcards.js.
